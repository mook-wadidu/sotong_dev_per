# 소통(Sotong) — 디자이너 사용법

> 외국인 손님이 QR로 자기 언어로 접수하면, 디자이너는 **한국어 요약**을 받고 응대합니다.
> 한 번만 설정해두면 그다음부터는 폰 알림 → 탭 몇 번으로 끝납니다.

---

## 0. 처음 한 번만 — 폰에 설치 + 알림 켜기 (가장 중요)

알림이 안 오면 대부분 이 단계를 안 한 것입니다. **순서대로** 해주세요.

### 아이폰 (Safari)
1. 사장님이 보내준 **내 인박스 링크**를 Safari로 엽니다. (`소통` 화면이 떠야 함)
2. 아래 **공유 버튼**(□↑) → **"홈 화면에 추가"** → 추가.
3. 홈 화면에 생긴 **소통 아이콘**으로 앱을 엽니다. *(Safari 탭이 아니라 홈 아이콘으로 여는 게 핵심)*
4. 화면 위쪽 **"새 손님 알림 · 알림 켜기"** → **허용**.
5. ✓ 표시가 뜨면 끝.
   - iOS 16.4 이상이어야 합니다. 안 되면 설정 → 일반 → 정보에서 버전 확인.

### 안드로이드 (Chrome)
1. 내 인박스 링크를 Chrome으로 엽니다.
2. 메뉴(⋮) → **"앱 설치"** 또는 **"홈 화면에 추가"**.
3. 설치된 **소통 앱**을 엽니다.
4. **"알림 켜기"** → **허용**. ✓ 표시 확인.

> ⚠️ **홈 화면에 추가만 하면 알림이 안 옵니다.** 반드시 앱을 열고 **"알림 켜기 → 허용"**까지 해야 손님 접수 알림이 폰으로 옵니다.
> 알림이 계속 안 오면 사장님께 **현재 내 인박스 링크**를 다시 받으세요(링크가 갱신됐을 수 있음).

---

## 1. 매일 쓰는 흐름 (한눈에)

```
폰 알림(새 손님) → 인박스 → 손님 요약 확인 → [상담 시작] 번역 채팅 → [시술 시작]
   → 시술 후 30초 기록(약제 + 비포·애프터 사진) → 손님에게 리포트 자동 발송
```

---

## 2. 인박스 (대기 손님)

- **내 손님**: 나에게 배정된 손님.
- **미배정**: 살롱 공용 QR로 들어온 손님. **"내 손님으로 가져오기"**를 누르면 내 손님이 됩니다(먼저 누른 사람이 가져감).
- 각 손님 줄에 **신규/재방문**, **대기/시술 중/완료** 상태가 표시됩니다.
- 새로고침으로 최신 목록을 봅니다(알림을 못 봤어도 여기서 확인).

## 3. 손님 요약 보기

손님 줄을 탭하면 **한국어 요약**이 뜹니다.
- 한 줄 요약 / 국적 / 원하는 시술 / 스타일 / **주의(알레르기 등)** / 예상 가격
- 손님이 올린 스타일 사진, 전체 상세 요약
- 재방문 손님이면 **지난 시술 기록**도 함께(지난번 약제·상태).

여기서 **[상담 시작]** → 손님과 번역 채팅, **[시술 시작]** → 시술 단계로 넘어갑니다.

## 4. 번역 채팅

- 디자이너는 **한국어로** 입력하면 손님 언어로 자동 번역돼 전달됩니다(반대도 자동).
- 자주 쓰는 말은 **칩(버튼)**으로 빠르게: 인사 / 응답 / 진행 / **가격** / **소요 시간** / 마무리.
  - 가격·시간 칩은 숫자를 고르면 손님 언어 표기로 나갑니다(원화가 손님 화면에 그대로 새지 않음).
- 칩에 없는 말은 직접 입력.

## 5. 시술 후 30초 기록 (★ 데이터가 쌓이는 곳)

시술이 끝나면 기록을 남깁니다. **비포·애프터 사진 2장이 필수**입니다.
- **사용 약제·제품**: 목록에서 탭(직접 추가도 가능).
- **모발 상태**: 상 / 중 / 하.
- **만족도**(선택): 펼쳐서 1~5.
- **비포 사진 + 애프터 사진**: 둘 다 찍어야 발송됩니다. *(후면 카메라로 시술 부위 촬영 — 얼굴 셀카 아님)*
  - 비포를 요약 단계에서 미리 찍어뒀으면 자동으로 채워집니다.
- **재방문 손님**이면 지난 약제·상태가 **기본값으로 미리 채워집니다** — 같으면 사진만 찍고 바로 발송.

**[종료하고 리포트 발송]** → 손님 언어 리포트가 자동 생성되고 링크가 뜹니다.

> 왜 사진 2장을 받나요? 비포/애프터가 "실제로 이렇게 됐다"는 객관 기록이라, 재방문 상담을 빠르게 하고 결과 데이터로 쌓입니다. 손님에게도 변신 기록(리포트)으로 돌아갑니다.

## 6. 손님 리포트

발송하면 손님은 **자기 언어 리포트**(시술 요약·약제·비포/애프터·홈케어·다음 방문 추천)를 받습니다. 재방문·리뷰로 이어집니다.

---

## 자주 막히는 곳

| 증상 | 해결 |
|---|---|
| 알림이 안 와요 | ① 앱(홈 아이콘)을 열고 "알림 켜기 → 허용" 했는지 ② iOS는 16.4+ ③ 사장님께 현재 인박스 링크 다시 받기 |
| "유효하지 않은 인박스 주소" | 옛 링크입니다. 사장님께 **현재 링크**를 다시 받으세요. |
| 리포트 발송이 안 눌려요 | **비포·애프터 사진 2장**을 모두 찍었는지 확인(둘 다 필수). |
| 손님 화면에 한국어가 보여요 | 손님은 자기 언어로 봅니다. 칩의 가격/시간은 손님 언어로 변환돼 나갑니다(정상). |

문의: 살롱 사장님 또는 소통 운영팀.
