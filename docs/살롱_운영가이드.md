# 소통(Sotong) — 살롱 운영 가이드 (사장님용)

> 외국인 손님이 **QR 하나**로 자기 언어로 접수하고, 디자이너는 **한국어 요약**을 받습니다.
> 사장님이 할 일: ① QR 비치 ② 디자이너 설정 ③ 외국인 손님에게 권하기.

---

## 1. QR 어디에, 어떻게 두나

목표: **외국인 손님이 들어와서 망설임 없이 찍게** 하는 것.

- **입구 / 안내 데스크**: "여기 찍고 자기 언어로 상담하세요" — 첫 접점.
- **대기 의자 / 거울 앞 / 테이블텐트**: 기다리는 동안 자연스럽게 접수.
- **명함·인스타·카톡 프로필**(디자이너 개별 QR): 단골·지명 손님이 미리 찍게.

비치물에는 **다국어 한 줄 안내**를 함께 두면 좋습니다(→ `손님_안내.md`의 카드 문구 사용).

### QR 두 종류
| QR | 용도 | 동작 |
|---|---|---|
| **살롱 공용 QR** (기본) | 워크인 외국인 — 어느 디자이너인지 몰라도 됨 | 찍으면 디자이너들에게 알림 → 먼저 받는 사람이 응대 |
| **디자이너 개별 QR** (선택) | 단골·지명 손님 | 그 디자이너에게 바로 배정 + 알림 |

> 디자이너가 1~2명인 살롱은 **공용 QR 하나로 충분**합니다. 개별 QR은 디자이너가 명함·SNS로 자기 손님을 끌어올 때 씁니다.

**QR 발급/인쇄**: 콘솔(사장 페이지) → **QR 탭**에서 공용 QR + 디자이너별 QR을 인쇄용으로 뽑습니다.

---

## 2. 외국인 손님에게 권하는 법 (현장 멘트)

말이 안 통해도 됩니다. **QR을 가리키며 폰을 가리키는 제스처**면 충분합니다.

- 영어: *"Scan this — order in your language."*
- 일본어: *"これを読み取ると、自分の言語で相談できます。"*
- 한국어(보조 직원용): "이거 찍으면 손님 언어로 상담돼요."

손님이 찍으면 → 언어 선택 → 탭으로 접수(1~2분) → 디자이너 폰에 알림.
**설치도 가입도 없습니다.** 손님은 그냥 찍고 탭만 합니다.

---

## 3. 처음 세팅 (온보딩)

1. **살롱 등록 + 사장 계정**: 소통 운영팀이 만들어 드립니다(콘솔 링크 전달).
2. **메뉴·가격**: 콘솔 → **메뉴 탭**에서 시술/카테고리/가격(직급별 가격 포함) 입력.
3. **디자이너 추가**: 콘솔 → **디자이너 탭**에서 추가 → 각 디자이너에게 **본인 인박스 링크** 전달.
4. **디자이너 폰 설정**: 각 디자이너가 폰에 앱 설치 + 알림 켜기 (→ `디자이너_사용법.md` 0번).
   - **현장에서 한 명씩 같이** 설치·알림 테스트까지 해두세요. iOS는 특히 빠지기 쉽습니다.
5. **QR 비치** + 손님 안내 카드.

---

## 4. 콘솔에서 보는 것 (사장 페이지)

- **메뉴 탭**: 시술/가격/직급별 가격 편집.
- **디자이너 탭**: 디자이너 추가/직급/링크.
- **QR 탭**: 공용 + 디자이너별 QR 인쇄.
- **문의/손님 탭**: 들어온 상담 목록, 손님별 시술 기록(카르테).

---

## 5. 출시 전 체크 (Launch Checklist 요약)

```
[ ] 살롱 QR 제작·비치 (입구·대기·테이블텐트 + 다국어 안내 카드)
[ ] 디자이너 전원 폰에 앱 설치 + 알림 켜기 + 테스트 1발
[ ] 메뉴·가격 입력 확인
[ ] 손님 1명 시뮬레이션: QR → 접수 → 디자이너 알림 → 응대 → 기록(사진 2장) → 리포트
[ ] 개인정보 동의가 접수 단계에 뜨는지 확인 (필수 + 학습용 선택)
```

## 자주 묻는 것

| 질문 | 답 |
|---|---|
| 디자이너 폰에 알림이 안 와요 | 앱(홈 아이콘)을 열고 **"알림 켜기 → 허용"**까지 했는지 / iOS 16.4+ / 현재 인박스 링크인지 확인. (→ `디자이너_사용법.md`) |
| 손님이 사진 찍히는 거 싫어하면? | 접수 단계에서 **동의**를 먼저 받습니다. 비포/애프터는 시술 부위(후면 카메라) 위주라 부담이 적고, 손님에겐 변신 기록 리포트로 돌아갑니다. |
| 중국어도 되나요? | 1차는 영어·일본어. 중국어는 다음 단계입니다. |
| 가격은? | 현재 파일럿(가격 미확정). 운영팀과 협의. |

문의: 소통 운영팀.
